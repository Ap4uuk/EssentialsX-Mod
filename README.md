## 🚀 Установка

1. Установите **Minecraft Forge 1.20.1 (47.+)**.
2. Скачайте последний релиз из раздела [Releases](../../releases).
3. Поместите JAR-файл в папку `mods/` сервера или клиента.
4. Запустите игру — мод активируется автоматически.
